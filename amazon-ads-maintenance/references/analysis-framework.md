# 运营分析框架

对飞书运营看板数据进行全面诊断时，按照以下框架逐项分析。

## 一、数据总览

汇总本次更新涉及的数据范围：

- 本次新增/更新的周范围
- 累计总销量、总销售额、总广告花费
- 本周期关键指标与上周期对比（环比变化）

## 二、销量与销售额趋势

### 分析维度

| 指标 | 计算方式 | 健康参考值 |
|---|---|---|
| 周均销量 | 总销量 / 周数 | >3 台/周 |
| 销量趋势 | 近 4 周 vs 前 4 周 | 持平或上升 |
| ASP 稳定性 | 各周 ASP 标准差 | 波动 <10% |

### 关注信号
- 连续 2 周以上零销量 → 🔴 严重
- 销量周环比下降 >50% → 🟡 警告
- ASP 大幅波动 → 检查是否有折扣活动影响

## 三、广告效率分析

### 核心指标

| 指标 | 公式 | 健康参考值 | 说明 |
|---|---|---|---|
| ACOAS | AD-spend / T-Sales | <15% | 广告成本占总销售比 |
| CPC | AD-spend / Click | <15 MXN | 单次点击成本 |
| AD-CVR | AD-order / Click | >3% | 广告转化率 |
| ROI | AD-Sales / AD-spend | >5 | 广告投资回报 |
| 广告订单占比 | AD-order / T-orders | 30%-70% | 过高说明依赖广告，过低说明广告无效 |

### 关注信号
- ROI < 1 → 🔴 广告亏损，每花 1 元广告费回不了 1 元销售
- 连续 3 周 AD-CVR = 0% → 🔴 广告完全无效，建议暂停
- CPC 持续上升 → 🟡 竞争加剧或关键词质量下降
- 广告订单占比 >80% → 🟡 过度依赖广告，自然排名弱

## 四、流量分析

### 核心指标

| 指标 | 健康参考值 | 说明 |
|---|---|---|
| T-Sessions | >100/周 | 总浏览量 |
| T-CVR | >5% | 整体转化率 |
| 自然流量占比 | Sessions - Click 估算 | 越高越好 |

### 关注信号
- T-Sessions < 50/周 → 🟡 流量极低，Listing 曝光不足
- 关闭广告后 Sessions 骤降 → 🔴 自然流量接近零
- T-CVR < 2% → 🟡 Listing 说服力不足，检查主图/价格/评论

## 五、成本与利润分析

结合费用表数据进行分析：

### 单件利润模型（基于费用表）

```
单件毛利 = 售价 × (1 - 平台扣点率) - 入库成本
单件净利 = 单件毛利 - 单件广告分摊
单件广告分摊 = 总广告花费 / 总销量
```

### 关注信号
- 单件广告分摊 > 单件毛利 → 🔴 卖一台亏一台
- 毛利率 < 15% → 🟡 利润空间薄，抗风险能力弱
- 广告花费 > 周销售额 × 20% → 🟡 广告占比过高

## 六、问题诊断与建议输出格式

### 问题按严重程度分级

- 🔴 **严重问题**：需要立即行动（如暂停亏损广告、处理产品质量问题）
- 🟡 **警告问题**：需要关注和计划（如优化 Listing、调整定价策略）
- 🟢 **正面信号**：值得保持和加强的趋势

### 建议按时间维度分类

1. **短期止血**（本周可执行）
   - 暂停/调整广告
   - 处理退货/差评
   - 紧急价格调整

2. **中期优化**（2-4 周）
   - Listing 优化（标题、主图、A+、关键词）
   - 广告策略调整（竞价、匹配方式、否定词）
   - 促销活动策划（结合节日节点）
   - Vine 评论获取

3. **长期策略**
   - 市场定位评估（价格带、目标客群）
   - 补货与库存规划
   - 站点投入产出评估（是否值得继续运营）
