# Pexar Influencer Evaluator

评估红人是否适合与 Pexar 数码相框合作（置换或付费），输出 **YES / MAYBE / NO** 三档结论。

## 品牌背景

- **品牌**：Pexar，Lexar 旗下子品牌，30年存储行业经验
- **核心产品**：11寸 2K 防眩光数码相框，搭载 Frameo App，32GB 存储，Wi-Fi 连接
- **产品定位**：送给家人（尤其是父母）的高品质科技礼物，强调情感连接与远程分享
- **目标市场**：Amazon 美国站，目标受众 25-45 岁有送礼需求的子女
- **核心卖点**：2K 防眩光屏、远程传照片/视频、大存储、触屏易操作、家居装饰感

## 硬门槛（任一不满足直接 NO）

在开始评分之前先检查：
- **粉丝数 < 10,000** → 直接 NO，不继续评估
- **内容非英语且面向非西方市场** → 直接 NO
- **纯口播、无生活场景画面** → 直接 NO（产品依赖视觉呈现）
- **过往合作品牌以廉价产品为主** → 直接 NO

## 理想红人画像

Pexar 目标受众是"为父母选礼物的美国子女"。最适合的红人是：
- **受众以精致、有消费力的美国女性为主**（25-45岁），偏白人/西方主流审美
- **内容有情感温度**：家庭、亲子、记忆、送礼，而非纯测评/性价比向
- **主页视觉高级**：精致生活方式，不杂乱，看起来有钱有品味
- 不要求一定是家居/科技博主，但内容里要有"家庭情感"基因

## 评估流程

当用户提供红人链接或信息时，按以下维度逐项评估，最终给出综合判断。

### Step 1：获取红人信息

优先使用 Chrome MCP 浏览器工具访问红人主页（截图+阅读页面内容），提取：
- 粉丝数、点赞总数、视频数量
- Bio 描述
- 近期5-10个视频的主题、播放量、评论氛围
- 整体主页视觉风格（截图观察）
- 合作过的品牌（从视频标题/描述/标签中找）

**⚠️ 重点排查竞品合作**：在搜集信息时，主动搜索该红人是否合作过以下品牌：
- **数码相框**：Skylight Frame、Aura Frame、Nixplay、Meural、Dragon Touch、Rollova
- **数码日历/家庭屏幕**：Skylight Calendar、Cozi、Smart Display 类产品
- 搜索方式：在其主页搜索品牌关键词，或用 WebSearch 搜索 `"@账号名" + "Skylight/Aura/Nixplay"`

如果工具无法获取（如 TikTok 反爬），请用户提供截图或口述关键信息。

### Step 2：六维评估打分

对每个红人按以下 6 个维度打分（每项 1-5 分）：

#### 2.1 视觉美学调性（★核心维度，权重最高★）

这是最重要的筛选维度。Pexar 是一款"有设计感的情感礼品"，视觉调性不对，数据再好也不合适。

| 分数 | 标准 |
|------|------|
| 5 | 画面精致高级，生活方式内容统一，像一本精选生活杂志 |
| 4 | 整体偏高端，视觉较统一，有品质感 |
| 3 | 内容尚可，但风格不够统一或不够高级 |
| 2 | 杂乱，视觉不精致，给人廉价感 |
| 1 | 风格完全不符（哥特/低俗/搞笑向/杂货铺式） |

**得分 ≤ 2 → 无论其他分数多高，直接 NO**

#### 2.2 情感向内容基因（★核心维度★）

Pexar 走情感路线，核心场景是"子女给父母传照片，父母随时看到孙辈成长"。需要红人有情感内容土壤。

| 分数 | 标准 |
|------|------|
| 5 | 家庭/亲子/送礼内容占比高，情感故事自然流露，粉丝反应真实有共鸣 |
| 4 | 有明显的家庭/回忆类内容，整体温情向 |
| 3 | 偶有家庭内容，但不是主线 |
| 2 | 内容几乎都是产品测评/性价比，缺乏情感 |
| 1 | 无情感向内容，纯功能/娱乐/流量内容 |

**得分 ≤ 2 → 付费合作直接 pass**

#### 2.3 受众画像匹配

- 理想受众：**精致有消费力的美国女性，25-45岁，偏白人/西方主流审美**
- 从评论区语言、互动语气判断受众画像（英语为主是基本要求）
- 评估受众消费力：合作品牌档次是重要信号（合作过 Temu/Amazon Basics 等廉价品牌 → 受众消费力存疑）

#### 2.4 数据表现

| 红人量级 | 互动率基准 |
|----------|-----------|
| Nano/Micro（1K-50K） | > 3-5% |
| Mid-tier（50K-500K） | > 2-3% |
| Macro（500K+） | > 1.5% |

- 警惕：粉丝多但评论少、评论全是 emoji 无实质内容（假粉嫌疑）
- 播放量/互动率是必要不充分条件，数据好但调性不对的直接 pass

#### 2.5 内容制作能力

- 视频/图片拍摄质量（产品是 2K 屏，内容画面要好看）
- 是否有真实使用场景 vs 纯念稿
- 内容形式丰富度（开箱、vlog、测评、场景展示等）
- 偏好有实景、有生活场景展示的内容形式，纯口播不够高级

#### 2.6 过往合作评估（含竞品排查 ⚠️）

**① 竞品合作排查（必查）**

主动检查是否合作过数码相框/数码日历品牌：

| 竞品类型 | 品牌 | 发现后如何处理 |
|--------|------|------------|
| 数码相框 | Skylight Frame、Aura Frame、Nixplay、Meural、Dragon Touch | 查看内容口碑方向——若正面推荐，说明受众匹配度高，但需确认是否有排他协议；若差评，对我们是加分项 |
| 数码日历/家庭屏 | Skylight Calendar、Smart Display 类 | 受众高度重合，是强信号——但同样需要确认是否有排他 |

**② 整体合作品牌档次**
- 合作过的品牌档次如何？廉价/杂货类品牌为主 → 付费 pass
- 合作内容是否真实自然，还是生硬念稿式广告？

**竞品合作结论逻辑：**
- 合作过竞品 + 内容正面 + 无排他 → 受众验证过，属于**加分项**，优先考虑
- 合作过竞品 + 有排他协议 → 暂时不可合作，记录跟进时间
- 从未合作过相关品类 → 中性，需要从其他维度判断受众匹配度

#### 2.7 节点适配性（如有特定 campaign）

- 该红人能否围绕指定节点（如母亲节）自然产出让粉丝共鸣的内容？
- **品牌调性匹配 ≠ 节点适配**：纯 home decor 博主适合日常合作但不一定适合母亲节节点

### Step 3：综合判断与 YES/MAYBE/NO 定级

**第一步：检查否决项**
- 粉丝 < 10k → **NO**
- 视觉美学（2.1）≤ 2 → **NO**
- 情感基因（2.2）≤ 2 → **NO**（或降为置换候选）
- 过往合作全是廉价品牌 → **NO**

**第二步：综合打分定级**

| 总分（满分35） | 结论 | 合作形式 |
|------|------|------|
| 28-35 | **YES** | 强烈推荐付费合作 |
| 22-27 | **YES** | 推荐付费合作 |
| 18-21 | **MAYBE** | 可付费，建议先置换试水 |
| 13-17 | **MAYBE** | 仅置换合作 |
| < 13 | **NO** | 不推荐 |

**特殊情况：**
- 调性好但节点不适配 → 归入"日常合作候选池"，注明适合的合作节点
